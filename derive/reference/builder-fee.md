# Builder Fee

On top of earning "API Broker Fees", builders can directly charge their users a builder fee (a.k.a `extra_fee`). This fee is charged  *on top* of exchange fees / API Broker fees / rebates and are directly sent to the builder Derive Wallet once every 4 weeks.

<br />

### Setup

1. Register for the API Broker program: [form](https://docs.google.com/forms/d/e/1FAIpQLSe7FD6NIk6fzT0tOoOsS6jlQP1wP0RUkdiAcxseKjQzmZxy-A/viewform?usp=preview) (see API Broker page).
   1. Registering for the broker program will automatically register you for the broker program and allow you to use the builder fee feature.
   2. You will generate fees and rebates even before you are fully approved for the broker program!
2. Fill in the `referral_code` and `extra_fee` fields when sending orders via `private/order`.
   1. `referral_code` is case sensitive
   2. `extra_fee` must be between 1,000 and 0.000001 USDC

<br />

# Viewing your rewards

* Both API Broker Fees and Builder Fees can be viewed in this dashboard: [https://pro.derive.xyz/broker](https://pro.derive.xyz/broker)
* Type in your referral code (case sensitive)

<br />

Extra fees (as well as the API Broker rewards) are distributed to the Derive Wallet you listed in the above form once every 4 weeks.
