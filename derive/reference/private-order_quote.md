# Order Quote

### Method Name

#### `private/order_quote`

Performs a "risk dry run" on an order, returning the estimated fee and whether the order is expected to pass.<br /><br />For market orders, the fill is assumed to happen at the supplied limit price. For accurate results the caller<br />should calculate an expected fill price from orderbook state and add a small buffer to it to account for latency.<br /><br />Should any exception be raised in the process of evaluating the order, a standard RPC error will be returned<br />with the error details.<br />Required minimum session key permission level is `read_only`
