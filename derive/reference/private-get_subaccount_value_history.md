# Get Subaccount Value History

### Method Name

#### `private/get_subaccount_value_history`

Get the value history of a subaccount.<br />Supported periods: 900 (15m), 3600 (1h), 86400 (1d), 604800 (1w).<br />Returns up to 1000 entries per request. If the time range exceeds 1000 \* period seconds,<br />the start is clamped forward to return the most recent 1000 entries.<br />Required minimum session key permission level is `read_only`
