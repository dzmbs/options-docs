# Create Subaccount

### Method Name

#### `private/create_subaccount`

Create a new subaccount under a given wallet, and deposit an asset into that subaccount.<br /><br />See `public/create_subaccount_debug` for debugging invalid signature issues or go to guides in Documentation.<br />Required minimum session key permission level is `admin`
